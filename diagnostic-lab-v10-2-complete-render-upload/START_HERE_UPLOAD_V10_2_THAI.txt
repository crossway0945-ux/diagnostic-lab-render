V10.2 — โปรเจกต์เต็มชุดพร้อมอัปโหลด GitHub และ Render

ไฟล์นี้รวมระบบทั้งหมดและการแก้หน้าเขียวไว้แล้ว ไม่ต้องนำ hotfix แยกไฟล์มาประกอบเพิ่ม

ชื่อโฟลเดอร์หลังใช้ Windows > Extract All:
diagnostic-lab-v10-2-complete-render-upload

ไฟล์ package.json ต้องอยู่ตรงนี้ทันที:
diagnostic-lab-v10-2-complete-render-upload/package.json

ห้ามมีโฟลเดอร์ชื่อ diagnostic-lab-v10-2-complete-render-upload ซ้อนอยู่ข้างในอีกชั้น

ค่า Render:
Root Directory: diagnostic-lab-v10-2-complete-render-upload
Build Command: npm install
Start Command: npm start

วิธีใช้:
1. ลบหรือย้าย ZIP V10/V10.1 เดิมออกก่อน เพื่อไม่ให้เลือกผิด
2. คลิกขวา ZIP V10.2 แล้วเลือก Extract All
3. อัปโหลดโฟลเดอร์ diagnostic-lab-v10-2-complete-render-upload ขึ้น repository diagnostic-lab-render
4. ตรวจใน GitHub ว่าเปิด diagnostic-lab-v10-2-complete-render-upload/package.json ได้
5. ใส่ค่า Render ตามด้านบน
6. กด Manual Deploy > Clear build cache & deploy
