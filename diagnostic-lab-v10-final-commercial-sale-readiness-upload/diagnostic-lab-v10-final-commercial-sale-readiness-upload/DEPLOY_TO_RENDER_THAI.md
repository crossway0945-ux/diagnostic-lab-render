# วิธี Deploy Diagnostic Lab V10 บน Render

ใช้ค่าต่อไปนี้เท่านั้นสำหรับแพ็กเกจ V10 ชุดนี้:

```text
Root Directory: diagnostic-lab-v10-final-commercial-sale-readiness-upload
Build Command: npm install
Start Command: npm start
```

ก่อน deploy ให้ตรวจใน GitHub ว่าไฟล์อยู่ที่:

```text
diagnostic-lab-v10-final-commercial-sale-readiness-upload/package.json
```

ถ้า path ไม่ตรง ให้แก้โครงสร้าง GitHub ให้ตรงกับชื่อโฟลเดอร์นี้ ห้ามเดาชื่อใหม่ ห้ามใส่ `.zip` และห้ามเว้น Root Directory ว่าง

อ่านขั้นตอนเต็มใน `DEPLOY_THIS_VERSION_THAI.md`
