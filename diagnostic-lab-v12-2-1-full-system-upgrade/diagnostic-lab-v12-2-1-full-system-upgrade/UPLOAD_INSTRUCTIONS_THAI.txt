คำสั่งอัปโหลด V12.2.1

1. แตกไฟล์ diagnostic-lab-v12-2-1-full-system-upgrade.zip
2. จะได้โฟลเดอร์เดียวชื่อ diagnostic-lab-v12-2-1-full-system-upgrade
3. อัปโหลดโฟลเดอร์นี้ทั้งโฟลเดอร์ไว้ที่หน้าแรกของ GitHub repository
4. ตรวจใน GitHub ว่ามี path นี้จริง:
   diagnostic-lab-v12-2-1-full-system-upgrade/package.json
5. ตั้ง Render ตามนี้:
   Root Directory: diagnostic-lab-v12-2-1-full-system-upgrade
   Build Command: npm install
   Start Command: npm start
   Health Check Path: /api/health
6. ตั้ง NODE_VERSION=22.16.0 และ OPENAI_MODEL=gpt-5.1
7. ตั้ง OPENAI_API_KEY, SESSION_SECRET และ ADMIN_SECRET ด้วยค่าจริงใน Render
8. กด Manual Deploy -> Clear build cache & deploy
