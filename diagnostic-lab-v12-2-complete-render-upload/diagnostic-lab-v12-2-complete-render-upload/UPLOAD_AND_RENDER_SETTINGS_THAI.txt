ชื่อ ZIP
diagnostic-lab-v12-2-complete-render-upload.zip

ชื่อโฟลเดอร์ที่ต้องอัปโหลดขึ้น GitHub
diagnostic-lab-v12-2-complete-render-upload

ค่า Render ที่ต้องกรอกตรงตัว
Root Directory: diagnostic-lab-v12-2-complete-render-upload
Build Command: npm install
Start Command: npm start
Health Check Path: /api/health

วิธีอัปโหลด
1. แตกไฟล์ ZIP
2. จะได้โฟลเดอร์ชื่อ diagnostic-lab-v12-2-complete-render-upload
3. ไป GitHub > Add file > Upload files
4. ลากโฟลเดอร์ diagnostic-lab-v12-2-complete-render-upload ทั้งโฟลเดอร์เข้าไป ห้ามเปิดโฟลเดอร์แล้วเลือกอัปโหลดเฉพาะบางไฟล์
5. ก่อน Commit ตรวจว่าหน้า GitHub แสดง path diagnostic-lab-v12-2-complete-render-upload/package.json
6. Commit การอัปโหลด
7. ไป Render > Settings > Build & Deploy แล้วตั้ง Root Directory เป็น diagnostic-lab-v12-2-complete-render-upload
8. ตั้ง Build Command เป็น npm install และ Start Command เป็น npm start
9. กด Manual Deploy > Clear build cache & deploy

ห้าม
- เว้น Root Directory ว่าง
- ใส่ชื่อ ZIP พร้อม .zip ใน Root Directory
- สร้างโฟลเดอร์ซ้อนชื่อเดียวกันสองชั้น
- อัปโหลด node_modules, .cache, .env หรือข้อมูล production
