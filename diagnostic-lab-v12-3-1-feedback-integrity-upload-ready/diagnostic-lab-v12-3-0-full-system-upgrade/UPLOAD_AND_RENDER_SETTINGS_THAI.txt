3. เปิดโฟลเดอร์ที่แตกแล้ว ต้องเห็น package.json, server.js, render.yaml ทันที
4. อัปโหลดโฟลเดอร์ diagnostic-lab-v11-7-bilingual-premium ไปที่ GitHub repository root เพียงครั้งเดียว
5. ตรวจใน GitHub ให้เปิด path นี้ได้จริง:
   diagnostic-lab-v11-7-bilingual-premium/package.json
