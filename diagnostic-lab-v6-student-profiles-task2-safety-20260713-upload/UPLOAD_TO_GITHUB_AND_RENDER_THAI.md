# อัปโหลดเวอร์ชันนี้

## GitHub

อัปโหลดโฟลเดอร์ทั้งก้อนเข้า repository เดิม โดยคงชื่อโฟลเดอร์นี้ไว้:

`diagnostic-lab-v6-student-profiles-task2-safety-20260713-upload`

ห้ามอัปโหลดเฉพาะไฟล์ข้างในแบบกระจัดกระจาย และไม่ต้องแก้ environment variables เพิ่ม

## Render

ไปที่ `Settings` > `Build & Deploy` แล้วตั้งค่า:

- Root Directory: `diagnostic-lab-v6-student-profiles-task2-safety-20260713-upload`
- Build Command: `npm install`
- Start Command: `npm start`

จากนั้นกด `Save Changes` และ `Deploy latest commit` หนึ่งครั้ง

ไม่ต้องแก้ API key, login, quota, custom domain หรือค่าอื่นที่ใช้งานอยู่แล้ว
